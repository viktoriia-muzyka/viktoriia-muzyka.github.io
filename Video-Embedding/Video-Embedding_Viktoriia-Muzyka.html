How to Embed a YouTube Video

Step 1: Find the Video

Navigate to the YouTube video you want to embed. In this example, I'll be embedding: A video about a journal to save you from infinite scrolling.

Step 2: Click "Share"

Below the video, click the "Share" button.

Step 3: Choose "Embed"

In the "Share" options, select the "Embed" option.

Step 4: Copy the Embed Code

YouTube will provide you with a code snippet. Copy this entire code.

Step 5: Paste the Code into Your HTML

In your HTML file, paste the copied code where you want the video to appear.
